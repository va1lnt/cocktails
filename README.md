**# cocktails**# Cocktails - коллекция рецептов коктейлей

[![Демо-сайт](https://img.shields.io/badge/Demo-Live-brightgreen)](https://va1lnt.github.io/cocktails/)

Интерактивная коллекция рецептов коктейлей с удобным поиском и фильтрацией.

## О проекте

Это веб-приложение позволяет:
* просматривать коллекцию рецептов коктейлей;
* искать коктейли по названию;
* фильтровать коктейли по ингредиентам;
* видеть детали рецепта (состав, способ приготовления, украшения).

Сайт доступен по адресу: [https://va1lnt.github.io/cocktails/](https://va1lnt.github.io/cocktails/)

## Функционал

* **Каталог коктейлей** - полный список доступных рецептов.
* **Поиск по названию** - мгновенный поиск нужного коктейля.
* **Фильтрация по ингредиентам** - подбор коктейлей по имеющимся компонентам.
* **Детали рецепта** - подробная информация о каждом коктейле:
  * список ингредиентов с количеством;
  * пошаговая инструкция приготовления;
  * рекомендации по подаче и украшению.

## Структура проекта

Проект состоит из следующих основных компонентов:

* `index.html` - главная страница приложения;
* `css/` - стили оформления;
* `js/` - JavaScript‑код приложения;
* `data/` - JSON‑файл с рецептами коктейлей;
* `images/` - изображения для коктейлей.

## Как использовать

1. Откройте сайт [https://va1lnt.github.io/cocktails/](https://va1lnt.github.io/cocktails/).
2. Используйте поле поиска, чтобы найти коктейль по названию.
3. Воспользуйтесь фильтрацией по ингредиентам, если хотите подобрать коктейли из имеющихся продуктов.
4. Кликните на любой коктейль, чтобы увидеть подробный рецепт.

## Разработка

Чтобы запустить проект локально:

1. Скопируйте репозиторий:  
   ```bash
   git clone https://github.com/va1lnt/cocktails.git
   ```
2. Откройте файл `index.html` в браузере.

## Данные

Все рецепты хранятся в формате JSON (`data/cocktails.json`). Каждый рецепт содержит:
* `name` - название коктейля;
* `ingredients` - список ингредиентов;
* `instructions` - инструкция по приготовлению;
* `garnish` - рекомендации по украшению;
* `image` - путь к изображению.
